# 文档总览

本目录按用途分层，方便外部读者快速定位信息。

## 目录结构

- `user/`：面向使用者的说明文档
  - `user-guide.md`：从部署到完整游戏流程
  - `functionality.md`：接口、状态机、安全机制
- `deployment/`：部署与链上证据
  - `deployment-guide.md`：Sepolia 部署步骤
  - `record.md`：真实交易记录（含 tx hash）
  - `deployment-evidence.md`：证据模板
- `internal/`：内部报告与维护文档
  - `update-report.md`：按近期 Git 变更整理的更新摘要
  - `report.md` / `report_zh.md` / `report.tex`：课程报告材料
- `project/`：原始作业规范（txt/pdf）
- `final_submit/`：提交归档（报告与归档版合约）

## 阅读顺序（推荐）

1. 根目录 `README.md`
2. `user/user-guide.md`
3. `user/functionality.md`
4. `deployment/deployment-guide.md`
5. `deployment/record.md`
